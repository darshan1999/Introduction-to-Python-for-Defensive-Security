# 🌐 Perimeter Exposure Validator

## Overview
This tool scans critical ports on an external-facing host to identify exposed services.

## Features
- Scans high-risk ports (FTP, SSH, Telnet, SMB, RDP)
- Identifies unintended exposure
- Lightweight and dependency-free

## Use Case
Used by defenders to validate perimeter security and detect misconfigured firewall rules.

## Why It Matters
Externally exposed services are often the first entry point for attackers.
