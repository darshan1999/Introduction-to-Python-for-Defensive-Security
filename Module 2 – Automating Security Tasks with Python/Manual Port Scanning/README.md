# 🛡 Manual Port Scanning – Defensive Use Cases

This module demonstrates how low-level Python socket scanning can be applied to real defensive security scenarios.

## Use Cases Implemented

1. Perimeter Exposure Validation
2. Internal Network Mapping
3. Security Control Validation
4. Incident Response (Lateral Movement Detection)
5. Asset Inventory Automation

## Key Concepts

- TCP socket connections
- connect_ex() behavior
- Timeout handling
- Structured JSON storage
- Defensive scanning logic

## Why This Matters

Manual scanning provides full control over network probing and allows defenders to:

- Validate exposure without relying on tools
- Build lightweight detection utilities
- Understand real TCP/IP behavior
